# Rock Paper Scissors – Java Game 🎮

A simple and fun **Rock-Paper-Scissors** game built in Java for beginners.  
The player competes against the computer for **5 rounds**, and the game shows the final winner at the end with a touch of humor! 😄



## 🎯 Features
- User vs Computer gameplay  
- 5-round game with final winner announcement  
- Fun and humorous ending messages 😅  
- Beginner-friendly Java code using simple **if-else statements**  
- Keeps track of **wins, losses, and draws**  



## 🛠️ Technologies Used
- **Java** (Core concepts, Random numbers, Loops, Conditions)
- **Scanner** for user input
- **Random** for computer moves
